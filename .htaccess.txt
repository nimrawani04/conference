#FILE NAME SHOULD BE [.htaccess]
This file should be inside Public_html (not inside dist) ,the steps are -
1.open h_panel that is hostinger -websites-> website list(we will find the hosted website)->dashboard(our websites dashboard)
							or
1.open h_panel that is hostinger -> home ->there if we scroll down we'll see our domain name ->click on manage (this will open our websites dashboard)
2.Go to File Manager → public_html.
3.Make sure .htaccess file exists (create a new one if not).
4.Paste the below code.
5.Save

THE CODE :

<IfModule mod_rewrite.c>
  RewriteEngine On
  RewriteBase /

  # Serve existing files or directories
  RewriteCond %{REQUEST_FILENAME} -f [OR]
  RewriteCond %{REQUEST_FILENAME} -d
  RewriteRule ^ - [L]

  # Redirect everything else to index.html
  RewriteRule ^ index.html [L]
</IfModule>

<IfModule mod_headers.c>
  # Always revalidate HTML so each visit checks hosting for the latest app shell
  <FilesMatch "\.(html)$">
    Header set Cache-Control "no-cache, no-store, must-revalidate"
    Header set Pragma "no-cache"
    Header set Expires "0"
  </FilesMatch>

  # Keep hashed build assets cacheable for performance
  # (new deploys generate new filenames, so users still get latest version)
  <FilesMatch "\.(js|css|png|jpg|jpeg|gif|svg|webp|woff|woff2|ttf)$">
    Header set Cache-Control "public, max-age=31536000, immutable"
  </FilesMatch>
</IfModule>
