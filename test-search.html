Search Functionality Test Guide

How to Test the Fixed Search Bar:

  1. Open the application at http://localhost:5173
  2. Desktop Search (visible on medium+ screens):
    • Click on the search bar in the top navigation
    • Type any of these test queries:
      • speaker - Should show keynote speakers
      • committee - Should show committee members
      • workshop - Should show workshop sessions
      • registration - Should show registration page
      • about - Should show about page
      • contact - Should show contact page
    • Click on any result to navigate to that page
    • Press Enter to go to the first result
  3. Mobile Search (visible on small screens):
    • Click the hamburger menu (☰) on mobile/small screens
    • Use the search bar at the top of the mobile menu
    • Type a search query and press Enter
    • Should navigate to the first matching result
